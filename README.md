## 📌 Overview

**Mini KFC** is a static landing page that captures the feel of a fast-food brand. It includes a sticky header, promotional banner, “Today’s Top Deals” section, and responsive layout — all without JavaScript. Perfect for practicing front-end layout or as a starting component for a larger project.

---

## ✨ Features

- 🧭 Sticky navigation: Sign In • Menu • Deals • Contact • Help  
- 🎯 Hero banner with promotional offer & call-to-action button  
- 🃏 “Today’s Top Deals” – 3 responsive deal cards with hover effects  
- 🎨 KFC-inspired color palette: red `#E31837`, cream `#FEF3E4`, golden `#FFC72C`  
- 📱 Fully responsive (mobile, tablet, desktop) using Flexbox/Grid  
- ⚡ Vanilla HTML & CSS – no frameworks, no JavaScript  
- 🔗 Placeholder links ready for real pages
