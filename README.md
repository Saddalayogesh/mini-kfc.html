📌 Project Overview

Mini KFC is a simple and clean landing page inspired by KFC.
Built using HTML & CSS, it focuses on layout, styling, and basic UI design.

---

🎯 Features

✔️ Clean header with KFC branding
✔️ Navigation menu (Signin • Menu • Deals • Contact • Help)
✔️ Banner with promotional offer
✔️ “Today’s Top Deals” section
✔️ Simple and neat styling

---

🛠️ Tech Stack

- 🌐 HTML5
- 🎨 CSS3
